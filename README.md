# scratchpad

Throwaway HTML demos and previews. Each demo lives in its own folder and is served via GitHub Pages at:

`https://sebastianlozano-owner.github.io/scratchpad/<slug>/`

## Demos

- [menu-direct-savings-footer-f](./menu-direct-savings-footer-f/) — Option F (E-strip evolution) from the menu direct-savings sketch.

## Adding a new demo

1. Create a new folder with a kebab-case slug: `mkdir <slug>`.
2. Drop `index.html` (and any local assets) inside.
3. Shared theme CSS lives at `/themes/default.css` (reference it as `../themes/default.css`).
4. Add a link to the root `index.html`.
5. Commit & push to `main`. The page goes live at `https://sebastianlozano-owner.github.io/scratchpad/<slug>/` within a minute.
